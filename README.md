# react-native-cloud-storage

Save & read from iCloud and Google Drive using React Native

## Installation

```sh
npm install react-native-cloud-storage
```

## Usage

```js
import { multiply } from 'react-native-cloud-storage';

// ...

const result = await multiply(3, 7);
```
